## Abstract

Περιγραφή των απαιτήσεων χρήση για τη δημιουργία διαδικτυακού e-shop πώλησης έργων τέχνων με την ονομασία "House Art". Αναλύονται εκτενώς οι απαιτήσεις των χρηστών με τη χρήση σεναρίων χρήσης. Ουσιαστικά μοντελοποιούνται οι απαιτήσεις και αποσαφηνίζονται τεχνικοί όροι, που είναι απαραίτητοι για την κατανόηση των απαιτήσεων από όλους τους εμπλεκόμενους χρήστες.
